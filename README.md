# 日本株 新高値更新銘柄レポート自動生成パイプライン

## 概要

J-Quants API を利用して日本株の株価データを取得し、
その日の新高値更新銘柄の分析レポートを自動生成するパイプラインです。
毎日決まった時間にパイプラインを実行し、LINEにレポートを通知します。

以下を自動で実行します。

- 株価データ取得
- 新高値更新銘柄抽出
- チャート生成
- PDF形式のレポート生成
- Cloudflare R2へのアップロード
- LINE通知
- GitHub Actionsによる定期実行
